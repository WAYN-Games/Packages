    Namespace WAYN.Games.Blob

    Classes

    BlobHashMapBuilder<TKey, TValue>

    Fluent builder for constructing a BlobMultiHashMap<TKey, TValue> stored in a blob asset. Accumulates key-value pairs, automatically rehashes when the load factor is exceeded, and produces an immutable BlobAssetReference<T> via CreateBlobAssetReference(Allocator).

    Structs

    BlobMultiHashMap<TKey, TValue>

    An immutable multi-value hash map stored in a BlobAssetReference<T>. Supports multiple values per key with O(1) average bucket lookup and binary search within buckets. Built using BlobHashMapBuilder<TKey, TValue> and intended for read-only access at runtime.

    KeyComputation

    Result of computing a bucket index for a key, containing both the target bucket index and the processed key hash.

    KeyIndex

    Stores metadata about a key's values within a hash map bucket. Tracks the key's hash code and the location and count of its associated values in the bucket's values array.

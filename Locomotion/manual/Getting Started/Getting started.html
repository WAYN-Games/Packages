    Installation

    Prerequisites

    • Read the full Getting Started doc before getting started
    • Unity 6+

    Import the package

    1. Open the package manager
    2. Install the package tarball

    Import the demo

    To get you started quickly and validate the installation, you can import one of the sample.

    1. Open the package manager
    2. Find the WAYN Locomotionpackage
    3. In the Samples tab of the package details select the sample to import :

    ⚠️ Every sample needs the SampleSharedResources to work properly.

    1. Import sample specific packages

    ⚠️ Some sample may need additional packages (Cinemachine, Input System, ...) to work properly. These packages are not included by default as dependencies of the Locomotion package to avoid cluttering your project with packages you might not use.

    1. Enter play mode. A capsule character will be spawned when a key is pressed.

    ℹ️ Refer to the Samples list to have an overview of what each sample is about

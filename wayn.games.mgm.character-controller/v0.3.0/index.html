    MGM Character Controller

    MGM Character Controller is a modular, high-performance character locomotion package for Unity 6, built entirely on the DOTS stack (Entities, Unity.Physics, Burst, Jobs). It provides a state-machine-driven approach to character movement with full source generation, making it both easy to extend and blazingly fast at runtime.

    Key Features

    • Pure ECS locomotion with Burst-compiled jobs for maximum throughput
    • State-machine-driven movement with a clear lifecycle per state
    • Source-generated boilerplate via Roslyn analyzers, so you focus on gameplay logic
    • Collide-and-slide physics with automatic step detection, slope handling, and ground snapping
    • Moving platform support with velocity inheritance and rotation handling
    • Hybrid GameObject layer for seamless integration with animations, audio, and cameras
    • Built-in debug visualization for contact points, movement paths, and collision impulses
    • Editor scaffolding tools that generate the entire state machine folder structure in one click

    移动平台

    MGM Character Controller 内置了对移动和旋转平台的支持。站在平台上的角色会自动继承其速度和旋转，无需在你的状态代码中编写任何特殊逻辑。

    工作原理

    移动平台是运动学物理体。运动系统检测角色何时站在任何动态物体（运动学的或完全模拟的）上，并自动计算角色应继承的位移和旋转。

    接触检测

    在 SnapToGround 步骤期间，系统检测到角色站在动态物体上。它将物体索引和实体引用记录在 LocomotionContact 组件中。

    InheritDisplacementFromMotionVelocity

    在运动作业结束时，InheritDisplacementFromMotionVelocity 步骤从物理世界读取物体的运动速度并将其分解为：

    • PlatformHorizontalVelocity - 物体速度投影到水平面上
    • PlatformVerticalVelocity - 物体速度沿重力轴

    位移在同一帧中应用于角色的位置。平台旋转通过扭转四元数计算直接应用。继承的速度在同一帧的 EvaluateTransition 步骤中立即可用（例如，跳跃状态的 OnEnterState 可以读取 PlatformHorizontalVelocity 以在跳离平台时保持动量）。

    Important

    你永远不应该在 OnIntegrateState 中手动添加平台速度。系统会自动处理。自行添加会导致双重应用，使角色在平台上移动两倍速度。

    平台组件

    PlatformMover

    存储基于轨道的平台移动的移动速度和当前路点索引。由 PlatformMovementSystem 更新。

    Rotor

    存储旋转平台的角速度。系统在每个固定更新中应用旋转。

    Track

    将平台运动约束在预定义路径上。平台循环经过路点。

    有关逐步设置指南，请参阅 配方：设置移动平台。

    PlatformMovementSystem

    PlatformMovementSystem 在 BeforePhysicsSystemGroup 中运行，根据 PlatformMover 和 Rotor 组件计算所需的线速度和角速度。它只写入 PhysicsVelocity 组件——Unity.Physics 然后在物理步中积分这些速度以更新平台的变换。

    角色在平台上的行为

    静止站立

    当在平台上静止时，角色随平台移动。如果平台旋转，角色的旋转由 InheritDisplacementFromMotionVelocity 步骤通过扭转四元数计算直接更新（参见示例的 IdleState.OnIntegrateState 中的旋转/滚动/滑动插值逻辑）。

    在平台上移动

    当在移动平台上行走时，角色自身的速度与平台的速度叠加。角色相对于平台表面移动。

    跳离平台

    当角色跳跃时，JumpingState.OnEnterState 将 PlatformHorizontalVelocity 添加到角色的水平速度：

    ctx.BaseContext.HorizontalVelocity += ctx.BaseContext.PlatformHorizontalVelocity;

    这保持了平台的动量，因此从快速移动的平台上跳跃会将角色推向平台移动的方向。如果没有这个，角色在空中会表现为"弹回"到静止状态。

    平台推挤（重叠解析）

    如果平台移动到角色内部（例如，上升电梯的天花板），SolveOverlaps 步骤会将角色推出。如果角色被困在两个表面之间（例如，电梯天花板和地板），重叠状态会经历 Push -> Pinch -> Crush 的转换。

    当角色被推入几何体时，碰撞滑动算法仍然可以将其推回，因为背面碰撞被忽略。这意味着被平台推入墙壁内部的角色可以通过远离墙壁移动来逃脱——胶囊投射不会与几何体的内表面碰撞。

    自定义平台行为

    你可以通过编写自己的系统来修改平台实体上的 PhysicsVelocity 组件来创建自定义平台移动。运动系统会自动处理角色站在任何动态物体上时的继承。

    Important

    你的系统必须在 BeforePhysicsSystemGroup 中运行，以便在 Unity.Physics 将速度积分到平台变换之前设置速度。

    例如，一个跟随样条线的平台：

    [BurstCompile]
[UpdateInGroup(typeof(BeforePhysicsSystemGroup))]
partial struct SplinePlatformSystem : ISystem
{
    [BurstCompile]
    public void OnUpdate(ref SystemState state)
    {
        float dt = SystemAPI.Time.fixedDeltaTime;
        foreach (var (velocity, spline, transform) in
            SystemAPI.Query<RefRW<PhysicsVelocity>, RefRW<SplineFollower>, RefRO<LocalTransform>>()
)
        {
            float3 targetPos = EvaluateSpline(spline.ValueRO, SystemAPI.Time.ElapsedTime);
            velocity.ValueRW.Linear = (targetPos - transform.ValueRO.Position) / dt;
        }
    }
}

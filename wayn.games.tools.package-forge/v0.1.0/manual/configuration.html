    Configuration

    All DocTools settings for a package live in a single file: Documentation/doctools.config. This file is a Bash script that is sourced during the build, so standard Bash syntax applies.

    Field Reference

    Required Fields

    Field Type Description
    PACKAGE_NAME string Package identifier (used internally and in metadata)
    PACKAGE_TITLE string Human-readable title shown in the navbar and page titles
    CURRENT_VERSION string The version that receives C# metadata extraction during builds
    ALL_VERSIONS array All versions to build, newest first. Example: ("0.3.0" "0.2.0" "0.1.0")

    Optional Fields

    Field Type Default Description
    PACKAGE_FOOTER string "" Footer HTML rendered at the bottom of every page
    METADATA_SRC string ".." Path (relative to Documentation/) to the directory containing C# source files
    LOGO_PATH string "" Path to logo image displayed in the navbar
    FAVICON_PATH string "" Path to the site favicon
    OUTPUT_DIR string "../Documentation~" Build output directory (relative to Documentation/)
    METADATA_ALLOW_COMPILATION_ERRORS string "true" Allow docfx metadata to succeed even when C# has compilation errors
    METADATA_NAMESPACE_LAYOUT string "nested" Namespace layout in generated API docs: "nested" or "flat"
    EXTRA_CONTENT array () Additional files to include in every version build (paths relative to Documentation/)
    RESOURCE_PATTERNS array () Glob patterns for non-markdown resources (images, etc.). Defaults to Manual/**/*.png if empty

    VERSION_CONTENT

    VERSION_CONTENT is a Bash associative array that lets you override content sources on a per-version basis. Declare it as:

    declare -A VERSION_CONTENT
VERSION_CONTENT["0.1.0"]="manual_src|manual_exclude|toc_src"

    Each value is a pipe-delimited string with three components:

    Component Default Description
    manual_src Manual Source directory for manual pages (relative to Documentation/)
    manual_exclude Draft/**,Draft V2/** Comma-separated glob patterns to exclude from the manual
    toc_src . Directory containing toc.md and index.md for this version

    Leave a component empty to keep the default. For example, to only override the exclude patterns:

    VERSION_CONTENT["0.2.0"]="|Internal/**|"

    Example

    A complete doctools.config for a Unity package with two versions:

    PACKAGE_NAME="com.wayngames.mypkg"
PACKAGE_TITLE="My Package"
PACKAGE_FOOTER="<span>&copy; 2025 WAYN Games</span>"
METADATA_SRC=".."
LOGO_PATH=""
FAVICON_PATH=""
OUTPUT_DIR="../Documentation~"
CURRENT_VERSION="0.2.0"
ALL_VERSIONS=("0.2.0" "0.1.0")

declare -A VERSION_CONTENT
VERSION_CONTENT["0.1.0"]="Manual||v0.1.0"

EXTRA_CONTENT=("../LICENSE.md" "../CHANGELOG.md")
RESOURCE_PATTERNS=("Manual/**/*.png" "Manual/**/*.gif")

METADATA_ALLOW_COMPILATION_ERRORS=true
METADATA_NAMESPACE_LAYOUT="nested"

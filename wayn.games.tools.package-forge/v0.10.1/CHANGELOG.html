    [0.1.0] — 2025-03-18

    Initial release of DocTools.

    Features

    • init command — Scaffold a Documentation/ directory with doctools.config, root toc.md, index.md, and a Manual/ folder with starter pages.
    • build command — Build documentation for a single version or all versions at once. Runs docfx metadata (for the current version) then docfx build with the custom template.
    • version command — Manage documentation versions: list, add, remove, set-current. Adding a version snapshots the current TOC and index.
    • snapshot command — Copy the current root toc.md and index.md into the active version directory.
    • doctools.config — Single-file configuration covering package metadata, version lists, per-version content overrides (VERSION_CONTENT), extra content files, and resource patterns.
    • Multi-version output — Each version builds into its own v{version}/ subdirectory with a generated versions.js and root redirect.
    • WaynGames.StaticToc.Extension template — Custom DocFX template extending statictoc with:
      • Version dropdown in the breadcrumb bar
      • Dark / light theme toggle with localStorage persistence and system-preference detection
      • Mermaid diagram rendering
      • Syntax highlighting via Highlight.js
      • WCAG AA–compliant color palette for both themes
      • Responsive layout with print stylesheet
      • Reduced-motion and forced-colors (Windows High Contrast) support
